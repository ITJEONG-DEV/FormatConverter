# bin/ — FFmpeg 실행 파일 위치

이 폴더에 아래 파일을 넣으면 프로그램이 우선적으로 사용합니다.
(없으면 시스템 PATH에서 찾습니다.)

- `ffmpeg.exe`
- `ffprobe.exe`

## 다운로드 (Windows)
1. https://www.gyan.dev/ffmpeg/builds/ 에서 `ffmpeg-release-essentials.zip` 다운로드
2. 압축 해제 후 `bin/ffmpeg.exe`, `bin/ffprobe.exe` 를 이 폴더로 복사

> 실행 파일은 용량이 커서 git 저장소에는 커밋하지 않습니다 (`.gitignore` 처리됨).
